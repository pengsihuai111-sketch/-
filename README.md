# 小升初数学题库管理系统 v4.0

## 系统架构

```
前端 (Vue 3 + Element Plus)  →  API (FastAPI)  →  MySQL 8.0
```

## 快速启动

### 1. 启动后端 (端口 8000)
```bash
cd backend
python run.py
```

### 2. 启动前端 (端口 5173)
```bash
cd frontend
npm install
npm run dev
```

### 3. 访问系统
浏览器打开 http://localhost:5173

## 系统功能

| 模块 | 功能 |
|------|------|
| 用户认证 | 注册、登录、JWT鉴权 |
| 题库浏览 | 搜索、筛选、查看题目详情 |
| 错题管理 | 录入错题、答题反馈、错题分类 |
| 练习单生成 | 每日练习、错题重做、专题练习、自定义 |
| 学情诊断 | 知识点掌握度、薄弱点识别、遗忘预警 |
| 会员中心 | 套餐购买、订单管理、支付模拟 |
