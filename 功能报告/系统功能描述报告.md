# 智数·小升初学情诊断系统 — 功能描述报告

**版本：** v4.0  
**技术栈：** Vue 3 + Element Plus + FastAPI + SQLAlchemy + MySQL  
**主题配色：** 绿白风格（参考作业帮品牌色 #10B981）  

---

## 一、系统概述

本系统专为**小升初数学备考**设计，提供从**题库管理 → 练习出题 → 错题追踪 → 学情诊断**的完整学习闭环。系统包含 7 大核心模块，覆盖题目录入、组卷练习、错题巩固、掌握度分析等全流程功能。

---

## 二、模块功能详述

### 1. 用户认证模块

| 功能 | 说明 |
|------|------|
| 用户注册 | 支持用户名、密码、邮箱、年级、真实姓名注册 |
| 用户登录 | JWT 认证，登录后返回 token |
| 路由守卫 | 未登录自动跳转登录页 |
| 会员标识 | 顶部栏显示当前会员等级（免费/基础/高级） |

**API 端点：**
- `POST /api/auth/register` — 注册新用户
- `POST /api/auth/login` — 用户登录
- `GET /api/auth/profile` — 获取当前用户信息

### 2. 题库管理模块

#### 2.1 题库浏览

| 功能 | 说明 |
|------|------|
| 分页查询 | 支持页码和每页数量参数 |
| 多维度筛选 | 按年级、分类、难度、知识点、关键词搜索 |
| 统计数据 | 首页展示总题数、已验证答案数、带图题目数、难度分布 |

**筛选维度：** 年级、知识分类、难度（基础/中等/挑战）、题目关键词

#### 2.2 题目录入

支持 **3 种录入方式：**

| 方式 | 说明 |
|------|------|
| **单题录入** | 手动填写题目信息，支持设置知识点、分类、题型、难度、来源等 |
| **图片识别** | 上传题目截图（≤20MB），通过 AI 自动识别题目文本 |
| **PDF 识别** | 上传 PDF（≤50MB），自动逐页转换并识别，支持批量导出 |

**题目元数据字段：** 知识点、知识分类、题型（填空/选择/计算/解决问题）、难度、来源学校、来源考试、考试年份、年级、押题标记（重点/难点/高频/经典）

#### 2.3 批量导入

支持一次性导入多道题目，自动生成题目编号或使用自定义 ID，导入结果返回成功/失败列表。

**API 端点：**
- `GET /api/questions` — 题库分页查询（支持多条件筛选）
- `POST /api/questions` — 录入单道题目
- `POST /api/questions/batch` — 批量导入题目
- `POST /api/questions/recognize` — 图片识别题目
- `POST /api/questions/recognize-pdf` — PDF 识别题目
- `GET /api/questions/stats` — 题库统计数据
- `GET /api/questions/knowledge-points/list` — 知识点列表（按分类分组）
- `GET /api/questions/categories/list` — 知识分类列表

### 3. 错题管理模块

#### 3.1 错题列表

| 功能 | 说明 |
|------|------|
| 分页浏览 | 支持分页查询，每页默认 15 条 |
| 错误类型标注 | 概念错误 / 计算错误 / 审题错误 / 方法错误 / 其他 |
| 掌握状态 | 已掌握 / 待巩固 标签区分 |
| 重做次数 | 显示每道错题的重做次数 |
| 操作 | 反馈（记录重做结果）、删除单条、批量删除 |

#### 3.2 错题录入

支持 **3 种录入模式：**

| 模式 | 说明 |
|------|------|
| **手动选择** | 从题库搜索并选择题目，支持分类/难度/关键词筛选 |
| **图片上传识别** | 上传错题图片或 PDF，AI 自动识别并匹配题库 |
| **自动匹配** | 识别后自动搜索题库中相似题目，支持一键关联 |

**录入附加信息：** 考试名称、考试日期、错误类型、备注

#### 3.3 错题反馈

对错题进行重做反馈：
- 标记做对/做错
- 选择错误类型
- 自动更新知识点掌握度
- 自动增加重做次数

#### 3.4 错题统计

- 总错题数、已掌握数、待巩固数
- 错误类型分布（百分比进度条）

**API 端点：**
- `GET /api/wrong-questions` — 错题列表（分页）
- `POST /api/wrong-questions` — 添加错题
- `POST /api/wrong-questions/recognize` — 图片识别错题（支持 PDF）
- `PUT /api/wrong-questions/feedback` — 错题反馈
- `DELETE /api/wrong-questions/{record_id}` — 删除错题

### 4. 练习单模块

#### 4.1 生成练习单

**4 种生成模式：**

| 模式 | 说明 |
|------|------|
| **每日训练** | 按知识点分组选题，支持设置每组题数和难度 |
| **错题重练** | 自动生成：2 道计算题 + 3 道原题重做 + 2 道举一反三 |
| **一周练习** | 学期模式：周一到周四每天一份练习单，基于掌握度和遗忘曲线智能选题 |
| **手动选题** | 直接指定题目 ID 列表 |

**智能选题特性：**
- 自动去重（同一天不出重复题）
- 支持按知识类别/知识点分组选题
- 题数自动估算时间
- 支持难度筛选（基础/中等/挑战）
- 每周练习基于掌握度分配薄弱/中等/掌握知识点

#### 4.2 练习单管理

| 功能 | 说明 |
|------|------|
| 列表查看 | 显示所有练习单，含名称、类型、题数、用时、状态 |
| 状态标记 | **已做/未做** 状态区分，已完成显示得分 |
| PDF 下载 | 生成学生卷 + 答案卷双 PDF，含分数渲染、题型分区、答题留空 |
| 预览题目 | 在线查看练习单全部题目 |
| 删除 | 单条删除和批量删除 |

#### 4.3 批改模式

**手工批改（对错标记）：**
- 逐题标记正确/错误
- 错误时可选择错误类型（概念/计算/审题/方法/其他）
- 统一提交后自动：
  - 计算得分
  - 更新知识点掌握度
  - 错题自动加入错题本

**自动批改：**
- 系统自动对比用户答案与正确答案
- 支持模糊匹配（去空格、标点、大小写）
- 返回逐题批改结果

#### 4.4 纸质版上传

支持拍照上传纸质练习单图片，保存到服务器。

**API 端点：**
- `POST /api/practice/generate` — 生成练习单
- `POST /api/practice/generate-week` — 生成一周练习单
- `POST /api/practice/generate-redo` — 生成错题重练
- `GET /api/practice/sheets` — 练习单列表
- `GET /api/practice/sheets/{sheet_id}` — 练习单详情（含题目）
- `DELETE /api/practice/sheets/{sheet_id}` — 删除练习单
- `GET /api/practice/sheets/{sheet_id}/download` — 下载练习单文本
- `POST /api/practice/sheets/{sheet_id}/complete` — 手工批改提交
- `POST /api/practice/sheets/{sheet_id}/submit` — 自动批改提交
- `POST /api/practice/sheets/{sheet_id}/upload` — 上传纸质版图片

### 5. 学情诊断模块

#### 5.1 诊断概览

| 指标 | 说明 |
|------|------|
| 知识点覆盖 | 已练习的知识点总数 |
| 平均掌握率 | 所有知识点掌握率均值 |
| 薄弱知识点 | 掌握率 < 60% 的知识点数量 |
| 遗忘风险知识点 | 距上次练习时间较长的知识点数量 |
| 诊断建议 | 综合建议：薄弱点提醒、遗忘风险提醒、趋势判断 |

#### 5.2 薄弱知识点诊断

- 列出每个薄弱知识点的掌握率、错误题数
- **主要问题分析**：针对每个知识点的常见错误原因
- **学习建议**：具体可操作的学习方法
- **解题要领**：关键解题步骤与方法

覆盖 20+ 个知识点的专属建议，包括：分数应用题、行程问题、工程问题、几何面积、圆柱与圆锥、立体几何、简便运算、解方程、比和比例、浓度问题、经济问题、鸡兔同笼、牛吃草问题等。

#### 5.3 错误分析

| 功能 | 说明 |
|------|------|
| 错误类型统计 | 各错误类型的次数和百分比（带颜色进度条） |
| 知识点错误热力图 | 按错误题数排序的知识点错误分布 |
| 薄弱点学习建议 | 环形进度图展示掌握率，含练习次数、错题数、建议 |
| 趋势数据 | 近 30 天练习量统计 |

#### 5.4 掌握度明细

| 功能 | 说明 |
|------|------|
| 掌握度表格 | 知识点、掌握度进度条、练习次数、正确题数、遗忘风险、上次练习日期、状态 |
| 多列排序 | 支持按掌握度、练习次数、上次练习日期排序 |
| 掌握度分布 | 良好/一般/薄弱 三段分布统计 |
| 趋势判断 | 稳步提升 / 总体向好 / 薄弱点较多 |

**API 端点：**
- `GET /api/wrong-questions/diagnosis` — 诊断报告
- `GET /api/wrong-questions/error-analysis` — 错误深度分析

## 七、系统特色

1. **AI 识别录入** — 支持图片和 PDF 两种格式的题目识别，自动提取题目文本
2. **错题自动归集** — 练习批改后错误题目自动加入错题本，无需手动操作
3. **智能组卷** — 基于掌握度和遗忘曲线的一周练习单自动生成
4. **知识点掌握度追踪** — 每次练习自动更新，60% 以下自动标记为薄弱点
5. **遗忘风险预警** — 根据距上次练习天数自动计算遗忘风险分数
6. **绿色主题 UI** — 参考作业帮品牌风格，清新简洁的绿白配色
7. **双模式批改** — 支持手工对错标记和自动答案比对两种批改方式
8. **PDF 导出** — 学生卷 + 答案卷双 PDF，含公式渲染和分区排版
