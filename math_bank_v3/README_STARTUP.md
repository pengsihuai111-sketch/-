# 新会话启动指南

**版本**: v3.0  
**适用对象**: 冯紫桐  
**目标**: 在新会话中传入文件即可独立运行核心功能

---

## 1. 必须上传的文件清单

新建会话时，请将以下文件/文件夹**全部上传**：

```
math_bank_v3/                    ← 整个文件夹（含子目录）
├── index/                        ← 5个索引文件
│   ├── master_index.json           ← 【必须】主索引
│   ├── knowledge_point_index.json  ← 【必须】知识点索引
│   ├── school_index.json           ← 【必须】学校索引
│   ├── difficulty_index.json       ← 【必须】难度索引
│   └── question_type_index.json    ← 【必须】题型索引
│
├── questions/                     ← 【必须】7个分片文件
│   ├── 几何.json
│   ├── 方程与应用.json
│   ├── 计算.json
│   ├── 数论.json
│   ├── 逻辑.json
│   ├── 基础.json
│   └── 其他.json
│
├── images/                        ← 【必须】47张几何图片
├── dedup/                         ← 【必须】查重索引
│   └── dedup_index.json
│
├── student/                       ← 【必须】学生状态
│   └── 冯紫桐_学习状态.json
│
├── wrong_questions/               ← 【必须】错题库
│   └── wrong_question_bank.json
│
└── v3_toolkit.py                  ← 【必须】操作工具库
```

**总计**: 63个文件，约1.3MB

---

## 2. 各功能运行状态

### ✅ 功能一：学情诊断（完全可独立运行）

**需要什么**: `student/冯紫桐_学习状态.json`

**你可以说**:
```
诊断报告
最近怎么样
看看掌握度
```

**系统会输出**:
- 当前各知识点掌握度（34个知识点）
- 薄弱点TOP5
- 遗忘风险知识点
- 已掌握知识点列表
- 学习趋势

**依赖**: 无外部依赖，纯数据文件即可运行

---

### ✅ 功能二：错题重做练习单生成（可运行，需人工复核答案）

**需要什么**:
- `wrong_questions/wrong_question_bank.json`（34道错题记录）
- `questions/*.json`（分片，获取原题原文）
- `images/`（错题中有几何题时需要原图）
- `v3_toolkit.py`（操作接口）

**你可以说**:
```
错题重做
重做错题
生成错题重做练习单
```

**系统会输出**:
- 考情分析页（这次考试各知识点得分率）
- 学生卷：34道错题原题（含几何图）
- 答案卷：每题答案 + 解析

**⚠️ 重要提醒**:
- 34道错题中，部分题目的答案已完善，部分仍标记为"需配合原图"或"待补充"
- 生成后请人工快速浏览答案卷，确认关键答案无误再给冯紫桐使用
- 几何题答案100%可靠（有原图支撑），文字应用题答案建议复核

---

### ✅ 功能三：错题完成反馈（完全可独立运行）

**需要什么**:
- `student/冯紫桐_学习状态.json`
- `wrong_questions/wrong_question_bank.json`

**你可以说**:
```
重做做完了
第1题对了第2题错了
反馈重做结果
```

**系统会**:
- 记录各题对错
- 更新知识点掌握度
- 标记"需重点加强"知识点
- 更新遗忘曲线复习计划
- 输出结果摘要

**依赖**: 无外部依赖

---

### ⚠️ 功能四：生成每周练习单（可运行，答案质量需把关）

**需要什么**:
- 全部文件（主索引 + 分片 + 图片 + 工具库）
- 当前日期（用于计算遗忘曲线）

**你可以说**:
```
生成本周练习单
下周4天
```

**系统会**:
- 加载主索引（~120KB）
- 按遗忘曲线筛选需复习的知识点
- 按薄弱点优先级选题
- 去重（排除最近出过的题）
- 确保每天1道几何题
- 生成4份练习单（学生卷 + 答案卷）

**⚠️ 限制**:
- 200题库中部分题目答案为AI生成，未完全人工验证
- 建议：首次使用时，生成练习单后快速复核答案卷中的关键答案
- 随着你逐步反馈答案正误，题库会越来越可靠

---

### ⚠️ 功能五：新题入库（可运行，需人工确认元数据）

**需要什么**:
- 全部索引文件 + 分片 + 查重索引
- `v3_toolkit.py`
- 用户上传的新考试PDF或图片

**你可以说**:
```
上传考试并生成
更新记录
只分析不生成
```

**系统会**:
- 解析PDF/图片中的题目
- 自动标注知识点和难度（AI推测）
- 查重（对比已有200题）
- 展示识别结果供你确认
- 确认后追加到对应分片

**⚠️ 限制**:
- PDF解析依赖扫描质量，可能需要人工校对
- 几何题需手动裁剪原图并命名保存
- 新题答案需补充（入库时可只存题目，答案后续完善）

---

### ⚠️ 功能六：举一反三/专题练习（可运行，题库有限）

**你可以说**:
```
专题：几何
练行程问题
出计算专题
```

**系统会**:
- 直接加载对应分片（如`几何.json`）
- 按掌握度动态选难度
- 生成专题练习单

**⚠️ 限制**:
- 当前200题中，某些知识点题目数量较少（如"牛吃草"仅1题）
- 专题练习可能重复出现同一道题（如果该知识点只有1-2题）
- 随着新题入库，专题练习会越来越丰富

## 3. 当前已知限制（诚实清单）

| 限制 | 影响 | 解决方式 |
|------|------|---------|
| 151/200题答案待完善 | 生成答案卷可能有误 | 你反馈时标记"答案有误"，我会修正 |
| 几何题以外的图片未上传 | 行程线段图、统计图表等无图 | 如有需要，后续补传 |
| 某些知识点题量少 | 专题练习/举一反三选择少 | 持续录入新题 |
| 简便运算题答案未逐题计算 | 答案可能是框架，非精确数值 | 优先验证这类题 |
| 无 references/ 目录 | 举一反三时缺少外部题库参考 | 可用现有200题内部变式替代 |

## 4. 新会话首次使用建议流程

### 步骤1：上传文件
将 `math_bank_v3/` 整个文件夹拖入新会话。

### 步骤2：验证基础功能
输入：
```
诊断报告
```
预期：系统输出冯紫桐当前掌握度概览。

### 步骤3：验证错题重做（关键测试）
输入：
```
错题重做
```
预期：系统生成含34道错题的练习单 + 答案卷。请快速浏览答案卷，告诉我是否有明显错误。

### 步骤4：验证练习单生成
输入：
```
生成本周练习单
```
预期：系统生成4天练习单。请检查第一天题目数量是否合理（7-10题），是否包含1道几何题。

### 步骤5：开始使用
如果以上三步输出正常，系统即ready，可以日常使用。
