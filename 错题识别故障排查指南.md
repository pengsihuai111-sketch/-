## 常见问题排查

### 1. 识别返回空结果
- 检查 .env 配置文件中的 API Key
- 确保图片清晰，包含完整题目
- 查看 backend.log 日志文件

### 2. 识别出题目但没有答案
- 已修复：优化了 prompt，明确要求 AI 生成答案
- 如果题目依赖图形，确保图片清晰

### 3. 识别速度慢
- 上传前压缩图片（推荐 1200px 以内）
- 使用 JPG 格式而非 PNG

### 4. API 调用失败
- 检查 API Key 是否有效
- 检查 API 配额是否用完
- 查看后端日志获取详细错误信息

## 优化建议

1. **图片质量**: 使用高清图片，确保文字清晰
2. **拍照技巧**: 正面拍摄，光线充足，避免阴影
3. **图片内容**: 题目完整，避免批改痕迹遮挡
