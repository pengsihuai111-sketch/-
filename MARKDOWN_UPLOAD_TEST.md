# Markdown 文件上传测试指南

## 测试环境
- 前端服务器: http://localhost:5173
- 后端服务器: http://localhost:8000
- 测试文件: test_questions.md

## 测试步骤

### 1. 登录系统
1. 打开浏览器访问 http://localhost:5173
2. 使用您的账号登录

### 2. 进入错题本页面
1. 点击左侧菜单的"错题本"
2. 确认页面标题显示为"错题本"（已修复乱码）

### 3. 测试 Markdown 文件上传
1. 点击"录入错题"按钮
2. 选择"文件上传"标签页
3. 点击上传区域或拖拽 `test_questions.md` 文件
4. 等待识别完成

### 4. 验证识别结果
应该识别出 3 道题目：
- 第1题：分数计算题（1/2 + 1/3）
- 第2题：应用题（小明买书）
- 第3题：几何题（长方形面积）

### 5. 确认录入
1. 检查识别的题目内容是否完整准确
2. 为每道题选择考试名称、错误类型等信息
3. 点击"确认录入"按钮
4. 验证题目是否成功添加到错题本列表

## 预期结果
- ✅ Markdown 文件成功上传
- ✅ 识别出 3 道题目
- ✅ 题目内容完整（包括题干、选项、答案、解析）
- ✅ 成功录入错题本

## 如果遇到问题
1. 检查浏览器控制台是否有错误信息
2. 检查后端日志输出
3. 确认文件格式是否正确（UTF-8 编码）
4. 确认文件大小不超过 10MB

## API 端点信息
- 端点: POST /api/wrong-questions/recognize-pdf
- 参数: 
  - file: 上传的文件（支持 .md, .markdown, .pdf）
  - use_markdown: 对于 Markdown 文件会自动处理
- 返回: task_id 和识别状态

## 测试文件内容
test_questions.md 包含 3 道小升初数学题，格式为标准 Markdown。
