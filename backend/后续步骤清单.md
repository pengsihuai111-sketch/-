## 2. 安装 Tesseract OCR（可选，用于扫描件）

1. 下载安装包：https://github.com/UB-Mannheim/tesseract/wiki
2. 安装到默认路径（如 `C:\Program Files\Tesseract-OCR`）
3. 添加到系统 PATH 环境变量
4. 下载中文语言包：chi_sim.traineddata


## 5. 前端适配（需要前端开发）

### 5.1 显示答案和解析

在错题详情页面添加：
```vue
<template>
  <div class="question-detail">
    <!-- 题目 -->
    <div class="question-text">{{ question.ai_question_text }}</div>
    
    <!-- 答案 -->
    <div class="answer-section" v-if="question.ai_answer">
      <h4>答案</h4>
      <div class="answer-content">{{ question.ai_answer }}</div>
    </div>
    
    <!-- 解析 -->
    <div class="solution-section" v-if="question.ai_solution">
      <h4>解析</h4>
      <div class="solution-content">{{ question.ai_solution }}</div>
    </div>
    
    <!-- 题目图片 -->
    <div class="question-images" v-if="question.question_image_urls">
      <h4>题目图片</h4>
      <img 
        v-for="(url, index) in JSON.parse(question.question_image_urls)" 
        :key="index"
        :src="url"
        class="question-image"
      />
    </div>
  </div>
</template>
```

## 7. 常见问题

### Q1: OCR 识别失败
**原因**: 未安装 Tesseract OCR
**解决**: 安装 Tesseract 或使用电子版 PDF

### Q2: 识别结果没有答案
**原因**: LLM 未生成答案
**解决**: 检查 LLM API 配置，查看日志

### Q3: 图片无法显示
**原因**: 图片路径错误或文件不存在
**解决**: 检查 `IMAGE_DIR` 配置和文件权限

### Q4: 数据库字段不存在
**原因**: 未运行迁移脚本
**解决**: 运行 SQL 迁移脚本

## 8. 性能优化建议

1. **图片压缩**: 提取的图片可以压缩以节省存储
2. **缓存**: 对相同 PDF 的识别结果进行缓存
3. **异步处理**: 大文件 PDF 使用后台任务处理
4. **批量识别**: 支持一次上传多个 PDF
