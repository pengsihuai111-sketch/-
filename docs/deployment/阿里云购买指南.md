1. **镜像选择错误**：

   - ❌ 1核2GB（勉强够用）
   - ✅ 2核4GB（推荐）

3. **硬盘类型**：
   - ❌ 普通云盘（慢）
   - ✅ SSD云盘（快）

## 🔑 购买后的操作

### 1. 获取服务器信息

**登录阿里云控制台**：
```
https://ecs.console.aliyun.com
```

**找到你的服务器**：
- 点击"轻量应用服务器"
- 查看服务器列表

**记录以下信息**：
- 公网IP：xxx.xxx.xxx.xxx
- root密码：（如果忘记可以重置）

### 2. 重置root密码（如果需要）

1. 在服务器列表中，点击"更多" → "重置密码"
2. 设置新密码（建议：大小写+数字+符号，至少12位）
3. 重启服务器使密码生效

### 3. 配置安全组

**开放必要端口**：

1. 点击服务器 → "防火墙"
2. 添加规则：
   - 端口：22（SSH）
   - 端口：80（HTTP）
   - 端口：443（HTTPS）
   - 来源：0.0.0.0/0（所有IP）

### 4. 测试连接

**Windows用户**：
```powershell
# 打开PowerShell
ssh root@你的服务器IP

# 输入密码
# 成功后会看到 Ubuntu 欢迎信息
```

**Mac/Linux用户**：
```bash
# 打开终端
ssh root@你的服务器IP

# 输入密码
```

## 💡 常见问题

### Q1: 找不到新用户优惠怎么办？

**A**: 可能原因：
1. 账号已经购买过ECS/轻量服务器
2. 活动已结束
3. 地域选择不对（某些地域没有优惠）

**解决方案**：
- 尝试更换地域
- 联系阿里云客服确认
- 或者选择腾讯云/华为云（也有新用户优惠）

### Q2: 轻量应用服务器 vs 云服务器ECS，选哪个？

**A**: 推荐**轻量应用服务器**

**对比**：
| 项目 | 轻量应用服务器 | 云服务器ECS |
|------|--------------|------------|
| 价格 | ✅ 便宜 | 贵 |
| 配置 | ✅ 够用 | 更灵活 |
| 管理 | ✅ 简单 | 复杂 |
| 适用 | ✅ 中小项目 | 大型项目 |

**你的项目选择**：轻量应用服务器完全够用

### Q3: 选择哪个地域？

**A**: 推荐顺序：
1. **华东1（杭州）** - 速度快，稳定
2. **华东2（上海）** - 速度快，稳定
3. **华北2（北京）** - 北方用户优先

**避免**：
- 海外地域（延迟高）
- 西南地域（相对较远）

### Q4: 带宽选多少？

**A**: 
- **3Mbps** - 够用（10-20个用户）
- **5Mbps** - 更好（20-50个用户）
- **200Mbps无限流量** - 最佳（不限用户）

**建议**：如果价格差不多，选200Mbps无限流量

### Q5: 需要购买数据盘吗？

**A**: **不需要**
- 系统盘50GB已经够用
- 数据盘会增加成本
- 后期不够可以扩容

### Q6: 购买时长选多久？

**A**: **1年**（推荐）

**理由**：
- 新用户优惠只有首年
- 1年后可以转移到其他优惠活动
- 或者续费时参加续费优惠活动
