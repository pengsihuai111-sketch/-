## 使用方法

### 前端使用（无需修改）

错题管理页面的图片识别功能会自动使用优化后的策略：

```javascript
// 前端代码无需修改，自动享受优化
recognizeWrongQuestionImage(formData)
  .then(res => {
    // 处理识别结果
    console.log('识别题数:', res.questions.length)
  })
```

### 后端API调用

```python
from app.utils.deepseek import recognize_questions
from app.utils.recognition_config import RecognitionStrategy

# 方式1：自动策略（推荐）
result = await recognize_questions(
    image_bytes,
    filename="test.jpg",
    strategy=RecognitionStrategy.AUTO  # 默认值
)

# 方式2：强制单阶段（快速模式）
result = await recognize_questions(
    image_bytes,
    filename="test.jpg",
    strategy=RecognitionStrategy.SINGLE_STAGE
)

# 方式3：强制两阶段（高精度模式）
result = await recognize_questions(
    image_bytes,
    filename="test.jpg",
    strategy=RecognitionStrategy.TWO_STAGE
)
```

## 策略选择规则

系统会根据图片尺寸自动选择最优策略：

```
极小图片 (<1000px)  → 两阶段（高精度）
中等图片 (1000-2000px) → 单阶段（快速）
大图片 (>2000px)    → 单阶段（快速）
```

**智能降级**：即使选择了两阶段，检测到3+题时也会自动切换到单阶段。

## 最佳实践

### 1. 图片质量建议
- ✅ 清晰、光线充足
- ✅ 分辨率 ≥ 1200x1200
- ✅ 题目完整、无遮挡
- ❌ 避免模糊、倾斜
- ❌ 避免过度裁剪

### 2. 截图技巧
- 包含完整题目（题号、题干、选项、图表）
- 一次截取1-5题（不要太多）
- 留出适当边距
- 使用系统截图工具（Win+Shift+S）

### 3. 识别后检查
虽然准确率已提升到90%，但仍建议：
- 检查题目文字是否完整
- 检查答案是否正确
- 检查知识点分类是否准确
- 有配图的题目需要手动上传图片

## 故障排查

### 问题1：识别速度仍然很慢
**可能原因**：
- 网络延迟（API调用超时）
- 图片过大（>5MB）
- 服务器负载高

**解决方法**：
- 压缩图片后重试
- 检查网络连接
- 稍后重试

### 问题2：多题识别不完整
**可能原因**：
- 题目被截断
- 图片模糊
- 题号不清晰

**解决方法**：
- 重新截图，确保题目完整
- 提高图片清晰度
- 确保题号清晰可见（1. 2. 3. 或 一、二、三、）

### 问题3：识别结果错误
**可能原因**：
- 图片质量差
- 手写内容干扰
- 批改痕迹过多

**解决方法**：
- 使用原始试卷图片（未批改）
- 如果是批改后的试卷，确保印刷体清晰
- 手动修正识别结果

## 技术细节

### 单阶段识别流程
```
上传图片 → AI一次性识别所有题目 → 返回结果
耗时：30-60秒（1次API调用）
```

### 两阶段识别流程
```
上传图片 → AI检测题目边界 → 逐个识别每道题 → 返回结果
耗时：60-180秒（1+N次API调用）
```

### 智能降级机制
```
选择两阶段 → 检测到3+题 → 自动切换单阶段 → 返回结果
避免过多API调用
```

## 常见问题

**Q: 为什么有时候识别很快，有时候很慢？**
A: 系统会根据图片大小和题目数量自动选择策略。单题或双题使用两阶段（较慢但准确），多题使用单阶段（快速）。

**Q: 可以手动选择识别策略吗？**
A: 前端暂不支持，后端API可以通过 `strategy` 参数指定。建议使用自动模式。

**Q: 识别后的答案准确吗？**
A: AI生成的答案准确率约85-90%，建议人工检查后再使用。

**Q: 支持哪些题型？**
A: 填空题、选择题、计算题、应用题等小学数学常见题型。

**Q: 可以识别手写题目吗？**
A: 目前主要针对印刷体题目优化，手写题目识别准确率较低。
